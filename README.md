# 📊 Expense Analysis Dashboard (Power BI)

## 🚀 Project Overview

This Power BI dashboard analyzes income, expenses, savings, and financial behavior across different occupations and city tiers.

The dashboard provides insights into spending habits, savings patterns, and disposable income distribution.

---

## 📁 Dataset Information

- 📄 20,000 Records
- 📊 27 Features
- 💰 Income & Expense Analysis
- 🏙️ City Tier Segmentation
- 👨‍💼 Occupation-Based Insights
- 📈 Savings Metrics

---

## ✨ Dashboard Features

- Income Analysis
- Expense Distribution by Category
- Savings Rate Analysis
- Occupation-wise Financial Summary
- Disposable Income Insights
- Interactive Filters

## 📊 Key Insights

- Rent is the largest expense category.
- Groceries are the second-largest expense category.
- Savings rates are relatively consistent across occupations.
- Disposable income is distributed almost evenly among occupation groups.
